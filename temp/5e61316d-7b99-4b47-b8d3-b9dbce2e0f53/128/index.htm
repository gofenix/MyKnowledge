1. 连接超时，首先找到git的安装目录，找到/etc/ssh/ssh_config文件

  2. 用notepad++打开这个文件，如下图：

  3. 把如下内容复制到ssh_config文件的末尾处：并记得保存

    Host github.com

    User git

    Hostname ssh.github.com

    PreferredAuthentications publickey

    IdentityFile ~/.ssh/id_rsa

    Port 443
